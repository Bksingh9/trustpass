# Onboarding Workflows

## Vendor Onboarding

1. Create or join vendor organization.
2. Complete organization basics.
3. Add primary contacts.
4. Select service categories and capabilities.
5. Upload required documents.
6. Add references and case studies.
7. Review checklist.
8. Submit verification request.
9. Receive review status and change requests.
10. Receive trust score and badges after approval.

## Admin Review

1. Open review queue.
2. Inspect vendor profile, contacts, documents, references, and case studies.
3. Review document metadata and signed preview URLs.
4. Pass, fail, waive, or mark checks not applicable.
5. Recalculate trust score.
6. Approve, reject, or request changes.
7. Generate notification and audit events.

## Buyer Evaluation

1. Search vendors by category, location, trust status, badge, capability, and team size.
2. Open buyer-safe trust profile.
3. Add vendor to shortlist.
4. Request clarification or additional document summary.
5. Track vendor response.
6. Move shortlisted vendors into external procurement workflows.
